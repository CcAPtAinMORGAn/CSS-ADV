TYPES
INLINE  = FOR SINGLE LINE ONLY = STYLE ATTRIBUTE
======    
    SYNTAX: <tag style="property:value;">text /<h1>
    EXAPLE : <H1 style="color:#ff00ff;">text /<H1>


INTERNAL / EMBEDDED = IN THE HEAD TAG, INSIDE STYLE TAG
========================
    SYNTAX: 
        selector/tag {
            property:value;
            key:value;
        }

    EXAMPLE:  
        h1 {
            color:#ff00ff;
        }

EXTERNAL / OUTSIDE / SEPERATE .CSS FILE 
========================
= PROFESSIONAL APPROCH, SEPRATE FILE WHICH DO NOT MIX N MATCH , USE LINK TAG
     SYNTAX:
